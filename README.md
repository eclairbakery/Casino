# Casino

A small economy bot written in Rust to replace EclairBot's worst economy ever.

Pretty much an unlisted fork of [EclairBOT-rs](https://github.com/eclairbakery/EclairBot-rs), a stripped-down version of it.

> [!WARNING]
> The bot is still in alpha

If you want to make a pull request, first check our [priority list](https://github.com/eclairbakery/Casino/wiki/TO%E2%80%90DO-page).
